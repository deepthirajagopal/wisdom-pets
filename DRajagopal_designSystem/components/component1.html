Navigation

The Nagivation Bar is an essential component in the design system. The Nav bar is used for all nagivation through the website and is repsonsive to screen size.

Example

This example of navigation is responsive. Adjust the screen size to demonstrate the responsive behaviors.

  • Work
  • Contact

Accessibility

The navigation bar is made to be responsive using Bootstrap; it should collapse into the hamburger menu at the medium breaking point (for best accessibility on tablet and mobile).Ensure accessibility by using a <nav> element or, if using a more generic element such as a <div>, add a role="navigation" to every navbar to identify it as a landmark region for users using assistive technologies.

Also include ARIA tags: aria-expanded="[true or false]" and aria-current="page" to help assistive technologies recognize which pages are active and which menus are expanded.

Usage

Use the responsive navigation bar in all pages for accessibility.

Code

 
          <nav class="navbar navbar-expand-lg navbar-light navshadow drnav">
              <div class="container-fluid">
                  <a class="navbar-brand" href="#">
                      <img src="../img/logo_color.png" width="40%" class="d-inline-block align-top" alt="DRajagopal logo"></a>
                      <button class="navbar-toggler custom-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
                          <span class="navbar-toggler-icon"></span>
                      </button>
                      <div class="collapse navbar-collapse justify-content-end nav-bg" id="navbarNav">
                          <ul class="navbar-nav">

                                  <a class="nav-link active navtext" aria-current="page" href="#">Work</a>
